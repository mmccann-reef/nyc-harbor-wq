# NYC Harbor Water Quality Dashboard

An interactive HTML dashboard for the NYC DEP Harbor Survey Program.  
**[Live demo →](https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/)**

## Features

- 🗺️ **Interactive map** — all 85+ monitoring stations, color-coded by the selected parameter's average value
- 📈 **Time series** — click any station to see its full measurement history
- 📅 **Flexible date range** — defaults to the last 10 years; extend back to 1909
- 📊 **Summary statistics** — N, Min/Max, Mean, Median, Std Dev, quartiles
- 📅 **Seasonality chart** — monthly average for the selected period
- 🔬 **NYS water quality standards** — automatic compliance check for Fecal Coliform, Enterococcus, and Dissolved Oxygen
- ⚖️ **Compare mode** — select two stations for side-by-side analysis and an overlay chart

## Quick Start

### Step 1 — Generate the data file

```bash
pip install requests
python fetch_data.py
```

This downloads all records from NYC Open Data (dataset `5uug-f49n`) and writes
`data.json` (~2–5 MB). It takes about 1–3 minutes depending on your connection.

> **Optional:** Get a [Socrata app token](https://dev.socrata.com/docs/app-tokens.html) (free) and add it to `fetch_data.py` as `APP_TOKEN = "your_token_here"` for higher rate limits.

### Step 2 — Preview locally (optional)

```bash
# Any static file server works
python -m http.server 8080
# Then open http://localhost:8080
```

> **Note:** Opening `index.html` directly as a `file://` URL will fail due to CORS. You need a local HTTP server as shown above.

### Step 3 — Deploy to GitHub Pages

1. Create a new GitHub repo (or use an existing one).
2. Copy `index.html` and `data.json` into the repo root (or a `docs/` folder).
3. Commit and push.
4. Go to **Settings → Pages → Source** and choose the branch/folder containing the files.
5. Your dashboard will be live at `https://YOUR-USERNAME.github.io/YOUR-REPO/`.

## Refreshing the Data

The NYC DEP dataset is updated periodically. To pull fresh data:

```bash
python fetch_data.py   # overwrites data.json
git add data.json
git commit -m "Refresh harbor water quality data"
git push
```

---

## Fallback (Live API Mode)

If `data.json` is not present, the dashboard automatically falls back to fetching
data live from the Socrata API. This works but is slower and subject to API rate
limits. For the best experience, commit `data.json` alongside `index.html`.

## Data Source

**NYC DEP Harbor Survey Program**  
[NYC Open Data — Harbor Water Quality (5uug-f49n)](https://data.cityofnewyork.us/Environment/Harbor-Water-Quality/5uug-f49n)

The NYC Department of Environmental Protection has monitored harbor water quality
since 1909 at up to 85 stations across New York Harbor. Parameters include
Dissolved Oxygen, Fecal Coliform, Enterococcus, Temperature, Salinity,
Turbidity, Chlorophyll a, and more.

**Water Quality Standards (NYS)**

| Parameter      | Bathing     | Fishing/Boating |
|----------------|-------------|-----------------|
| Fecal Coliform | ≤ 200 MPN/100mL | ≤ 1,000 MPN/100mL |
| Enterococcus   | ≤ 35 MPN/100mL  | —              |
| Dissolved O₂   | ≥ 5.0 mg/L      | —              |
